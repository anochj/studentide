## What you will learn

By the end of this module, you will know how to:

- showing messages in the console
- run a Java source file
- change starter code safely

## How to run this module

From this course folder, run:

```bash
javac 03_printing_and_output/Main.java
java -cp 03_printing_and_output Main
```

Then press Enter.

## What this module is about

Showing messages in the console is one small part of programming. You will see the idea in the README first, then in `Main.java`.

## Common mistakes

Here are a few things to check if something breaks:

- spelling of names
- matching quotes or brackets
- missing semicolons when the language uses them
- running the command from the right folder

Errors are normal. They are part of learning.
