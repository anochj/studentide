- run it again and see the difference

## A few useful words

Here are a few words you will see often:

- **IDE**: Integrated Development Environment. This is the app you use to write and run code.
- **Python**: A popular programming language that is beginner friendly and widely used.
- **Console**: A text area where your program can show output.

## Open the file

Open [__main__.py](__main__.py).

This is the file you will use for this module.

Inside, you should see this code:

```python
print("Hello, World!")
```

This program tells Python to display the text `"Hello, World!"` in the console.

> Do not be afraid to experiment. This is a virtual machine, so if something goes wrong, you can always reset it.

## Your first Python program

The line below is one of the simplest Python programs you can write:

```python
print("Hello, World!")
```

The `print()` function tells Python to show something in the console.

In this case, it shows:

```
Hello, World!
```

## Run your program

To run Python code, open the terminal and use this command:

```bash
python3 01_welcome_to_python
```

Follow these steps:

1. Click **View** in the top menu bar.
2. Click **Terminal**.
3. In the terminal, type:

```bash
python3 01_welcome_to_python
```

4. Press Enter.

You should now see:

```
Hello, World!
```

Nice. You just ran your first Python program.

> Small note: this works because Python can run a folder when that folder contains a file named `__main__.py`. You do not need to worry about that yet.
