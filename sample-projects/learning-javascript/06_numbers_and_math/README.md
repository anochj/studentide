## What you will learn

By the end of this module, you will know how to:

- use numbers and simple math
- run a JavaScript file with Node
- change starter code safely

## How to run this module

From the `learning-javascript` folder, run:

```bash
node 06_numbers_and_math
```

Then press Enter.

## Open the file

Open [index.js](index.js).

Run it first. Then change one small value and run it again.

## Small example

```javascript
console.log("Hello from JavaScript.");
```

## Common mistakes

Here are a few things to check if something breaks:

- missing quotes around text
- missing closing brackets or parentheses
- spelling a variable name two different ways
- running the command from the wrong folder

Errors are normal. They are part of learning.
